Loja DEISI

Esta API disponibiliza vários endpoints da aplicação DEISI Shop.

Base URL: https://deisishop.pythonanywhere.com/[endpoint]

Endpoint: categories/

Método HTTP: GET

Retorna as categorias existentes na aplicação

Exemplo de pedido

https://deisishop.pythonanywhere.com/categories/

Exemplo de resposta

[
  "T-shirts",
  "Canecas",
  "Meias"
]

Endpoint: products/

Método HTTP: GET

Retorna os produtos existentes

Exemplo de pedido

https://deisishop.pythonanywhere.com/products/

Endpoint: buy/

Método HTTP: POST

Recebe uma lista de ids de produtos para comprar, assim como informação se é estudante, um cupão e um nome. Retorna o preço final, assim como uma referencia para pagamento

Exemplo de pedido

https://deisishop.pythonanywhere.com/buy/
Tendo no body 
{
    products: [1, 2, 3],
    student: true,
    coupon: black-friday,
    name: Maria Lisboa,
}

Exemplo de resposta

{
    cost: 23.43,
    reference: "201124-0004",
    message: "Maria Lisboa, recebemos a sua encomenda!",
    error: "",
}
